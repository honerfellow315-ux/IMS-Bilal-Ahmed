# 📦 Inventory Management System

A ready-to-install desktop Inventory Management System for Windows.
No coding or Qt installation required — just download and run the `.exe`

**Developer:** Bilal Ahmed 

---

## ⬇️ Installation

**Step 1 — Download**
```
Click on InventorySystem_Setup.exe in this repository
Then click the Download button
```

**Step 2 — Run the Installer**
```
1. Double-click InventorySystem_Setup.exe
2. If Windows SmartScreen warning appears:
   - Click "More info"
   - Click "Run anyway"
3. Installation wizard will open
```

**Step 3 — Install**
```
1. Click "Next"
2. Choose installation folder
   Default: C:\Program Files\InventorySystem
3. Click "Install"
4. Wait for installation to complete
5. Click "Finish"
```

**Step 4 — Launch**
```
Double-click the InventorySystem shortcut on your Desktop
```

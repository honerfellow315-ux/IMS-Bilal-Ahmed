PROJECT OVERVIEW:
  A desktop Inventory Management System built with C++ and Qt framework.
  Uses SQLite for local database storage.

LOGIN CREDENTIALS:
  Username : bilalahmed
  Password : Ahmed@0001

FEATURES:
  1. Secure Login System
  2. Product Management (Add / Edit / Delete / Search)
  3. Low Stock Alert System
  4. Reports with Export to File (.txt)
  5. Real-time Dashboard Statistics

TECHNOLOGIES USED:
  - Language  : C++ (Standard 17)
  - GUI       : Qt 6 Framework
  - Database  : SQLite (via Qt SQL module)
  - Build     : CMake

HOW TO RUN:
  1. Open Qt Creator
  2. Open CMakeLists.txt
  3. Build -> Rebuild All
  4. Press Play (green button)

FILE STRUCTURE:
  main.cpp           - Application entry point
  database.h/cpp     - Database initialization & queries
  loginwindow.h/cpp  - Login screen
  dashboard.h/cpp    - Main dashboard
  productmanager.h/cpp - Product CRUD operations
  lowstockalert.h/cpp  - Low stock detection
  reportsscreen.h/cpp  - Reports & export
  CMakeLists.txt     - Build configuration
